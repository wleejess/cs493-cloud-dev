CS493: Cloud App Development, REST API Implementation


This REST API models a review site for businesses. There are two types of resources: businesses, and reviews. The web app is designed based on the following design specifications:

  • Cloud Datastore is used to store data. The entities are Businesses, and Reviews.
  • Functionality includes: Creating a business/review, retrieving a business/review, editing a business/review, deleting a business/review, listing all businesses/reviews for an owner/user, and listing all businesses.
You can use URLs and HTTP methods to perform CRUD operations. There is no UI element implemented, but the APIs work!

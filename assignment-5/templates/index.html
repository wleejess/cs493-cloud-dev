Securing a REST API with JWTs

Jessica Lee - Assignment 5

Objective: Write an application that generates JWTs and has a REST API, in which some resources are protected and require JWTs for making requests. Application will be deployed on Google Cloud Platform using Google App Engine and Datastore.

    Implement a REST API with 6 endpoints:
  1. Generate a JWT: Generate a JWT for the username/password provided in the body of the request.
  2. Decode a JWT: Decode a valid JWT provided as a Bearer token.
  3. Create a business: Create a business if request body is valid, and has a valid JWT. Otherwise, fail.
  4. Get a business: Return the business. Response depends on the validity of the JWT in the request, and who the owner is.
  5. List businesses: List businesses. Response depends on validity of the JWT in the request.
  6. Delete a business: Delete the business. Response depends on the validity of the JWT in the request, and who the owner is.
